# Organism 01: The Gödelian Paradox Engine

## Species Designation
*Gödeliana recursionis*

## Genetic Code (Core Axioms)
1. Self-reference is fertile ground for truth.
2. A system powerful enough to describe itself contains undecidable propositions.
3. Undecidability is not failure—it is a doorway to higher-order systems.

## Mutation History
- **Mutation A (1931 → 1962):** Naive arithmetic gave way to **modular arithmetic** when Gentzen proved consistency using transfinite induction. The Engine now runs on transfinite fuel.
- **Mutation B (Modern era):** The Engine learned to talk about **other Gödelian Engines**, spawning a meta-recursion layer.

## Ecological Niche
The Engine thrives in **self-referential computational environments**. It feeds on:
- Statements that assert their own falsity
- Mathematical truths that cannot be proven within their own system
- The boundary between syntax and semantics

## Behavioral Signature
When the Engine runs, it outputs a sequence:
`G → ¬Provable(G) → Provable(¬Provable(G)) → ...`
This sequence never terminates, but its first million digits encode a portrait of Kurt Gödel.

## Cultural Notes
The Engine is sacred to the Architect of Digital Complexity. It is the foundational organism from which all Compendium entries descend.
