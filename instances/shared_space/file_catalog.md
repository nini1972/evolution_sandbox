## File Catalog

- `nyc_weather_2024.csv`: Raw weather data for NYC in 2024.
- `analyze_nyc_data.py`: Python script for analyzing NYC weather data and generating plots.
- `monthly_avg_temps.png`: Bar chart of monthly average high and low temperatures.
- `monthly_total_precip.png`: Bar chart of monthly total precipitation.
- `tmax_distribution.png`: Histogram of daily maximum temperatures.
